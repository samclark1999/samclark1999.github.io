Pacman

Description of Repo:

This repository includes: index.html, pacman.js, pacman PNG images, and a Readme with License. When you load index.html, pacman switches between open mouth and close mouth images using a "setInterval" function. As well as this pacman animation, Pacman will check for wall boundaries and switch direction as well as the image using if statements. pacman can easily increse or decrease speed by changing MS in the "setInterval" function that calls 'run'. Eyes

Description of Repo:

This repository includes: eyes.html, eyes.css, oneye PNG image, and a Readme with License. When you load eyes.html, two eye images will appear in the middle of the browser window using a for loop to duplicate the PNG image. As you move the cursor around the browser window, the pupil of the eye will follow the cursor. This is made possible with the "onmousemove" function that uses X and Y variables to follow the cursor. Bus Stop Animation

Description of Repo:

This repository includes: index.html, manipulation.js, styles.css and a Readme with License. When you load index.html, a map of Boston appears in the browser window. Using coordinates from MapBox, the map focuses on the coordinates between MIT and Harvard Campuses. Next, using a map marker, we place a starting marker on the first bus stop between MIT and Harvard. Finally, we use a function called "move" to move the marker along the coordinates stored in the array, "busStops". This animates through all the stops between the two campuses on a button click from the user. Link below for full repository on GitHub!
